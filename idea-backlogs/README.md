# AEP CLI Idea Sandbox

Adobe Experience Platform CLI의 발전을 위한 아이디어 저장소입니다.

## 🤖 에이전트 중심의 진화 (Agent-First)
- **AEP 마이그레이션 코파일럿**: 커맨드 입력 없는 대화형 마이그레이션 가이드
- **슈퍼바이저 에이전트 (Supervisor Agent)**: 자연어 의도를 CLI 커맨드 시퀀스로 변환 및 오케스트레이션
- **AEP 마인드맵 (AEP Mindmap)**: 복잡한 AEP 리소스(스키마, 데이터셋, ID) 간의 관계를 시각화

## 📋 기능 백로그 (Backlog)
- 기타 기능 제안: 마켓플레이스, 온보딩, 버전 관리 등
