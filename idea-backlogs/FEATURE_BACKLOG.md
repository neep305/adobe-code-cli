# Feature Backlog

AEP CLI에 추가될 수 있는 다양한 기능 아이디어 모음입니다.

## 🛒 템플릿 및 배포
- **템플릿 마켓플레이스**: 커뮤니티나 공식 저장소에서 템플릿을 직접 다운로드 (`adobe aep schema template import <url>`)
- **버전 관리**: 템플릿에 `version` 필드를 추가하여 업데이트 알림 및 마이그레이션 가이드 제공

## 🎓 사용자 경험 (UX)
- **대화형 온보딩**: `adobe onboarding start` 명령으로 진행되는 인터랙티브 위저드 모드
- **커스텀 체크리스트**: 사용자/조직별 목표에 따라 체크리스트 항목을 추가/제거할 수 있도록 설정 파일 지원?
- **Interactive TUI (Terminal User Interface)**: 방향키와 엔터로 선택하는 대화형 메뉴 시스템 도입

## 🛠️ 유틸리티
- **스키마 시각화**: 로컬에서 정의한 XDM 구조를 Mermaid나 시각적 도구로 즉시 확인
- **Dry-run 강화**: 실제 API 호출 없이 시뮬레이션 결과를 상세 리포트로 제공
