# AEP 마이그레이션 코파일럿 (Migration Co-pilot)

## 개요
사용자가 일일이 CLI 커맨드를 조합하여 데이터를 마이그레이션하는 방식은 실패율이 높습니다. 
에이전트가 데이터 소스 진단부터 스키마 매핑, 수집 확인까지 전체 과정을 대화형으로 리드합니다.

## 핵심 기능
1. **대화형 워크플로우 (Guided Experience)**
   - "사용자님, 업로드하신 데이터에는 'Email'이 있네요. 이를 Primary Identity로 설정할까요?"
   - 사용자는 (Y/N) 혹은 간단한 대답만으로 복잡한 설정을 완료
2. **지능형 스키마/ID 자동 추천**
   - 소스 필드와 AEP 표준 필드 간의 매핑 Confidence Score 제공
3. **자가 치유(Self-healing) 파이프라인**
   - 데이터 수집 중 형식 오류 발생 시 AI가 Transformation 로직 수정 제안
4. **커맨드 추상화**
   - 백엔드에서는 수많은 `aep schema`, `aep dataset` 커맨드가 돌지만, 사용자는 최종 결과만 확인

## 기대 효과
- **학습 곡선 제거**: CLI 파라미터를 공부할 필요 없음
- **오류 감소**: AI가 사전에 스키마 호환성을 검증
- XDM 표준 준수율 향상
- 데이터 엔지니어의 반복 작업 최소화
