🔧 Signup Functionality Diagnostic Tool

This tool helps diagnose issues with the web application signup functionality.

Test 1: Backend Health Check

Verify that the backend API is running and accessible.

Test 2: CORS Configuration

Check if CORS is properly configured for cross-origin requests.

Test 3: Registration API Test

Test the registration endpoint with actual data.

Test 4: Login Test

Test login with credentials (use the same email/password from registration test).
